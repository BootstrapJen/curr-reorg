Glossary

  • circle of evaluation: a diagram of the structure of an expression (arithmetic or code)

  • definitions area: the text box in the Editor, where definitions for values and functions are written

  • evaluates to: the result of computing an expression

  • function: a mathematical object that consumes inputs and produces an output

  • interactions area: the text box in the Editor, where we enter expressions to evaluate

  • mathematical expression: a computation in arithmetic

  • programming language: a set of rules for writing code that a computer can evaluate

  • value: a specific piece of data, like 5 or "hello"

Section 1

Welcome to Bootstrap! In this course, you’ll be learning a new programming language — a way to tell computers exactly what you want them to do.

Math is a language, just like English or Spanish. Where human languages use nouns (e.g., “bread”), math uses values.

Where human languages use verbs (e.g., “throw”), math uses functions.

A mathematical expression is like a sentence.

One way to indicate the order of operations in an expression is to first use a diagram called the Circle of Evaluation.

Every Circle of Evaluation evaluates to the result of its corresponding expression.

The Bootstrap screen has a Definitions area and an Interactions area.
