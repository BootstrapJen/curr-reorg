Glossary

  • apply: use a given function on some inputs

  • average: A number representing the center, or 'typical' value in a set of data

  • axis: A number line on a scatter plot, used to place points that represent pairs of values in data sets

  • conditional: a code expression made of questions and answers

  • data block: Code that lists the name, constructor(s), and field(s) of a data structure

  • data science: The study of using data to answer questions about the world

  • dataset: a collection of related information that is composed of separate elements, but can be manipulated as a unit by a computer

  • value: a specific piece of data, like 5 or "hello"
